Ustawienia

Bez logowania działasz w trybie demo — dane są zapisywane tylko na tym urządzeniu (usunięcie PWA = utrata danych). Konto włącza synchronizację (Cloudflare D1) i powiadomienia push. Konta zakłada właściciel instancji — zobacz README na GitHubie, jak postawić własną.

—

Klucz z themoviedb.org → Ustawienia → API (Read Access Token, nie api_key). Używany gdy serwer nie ma skonfigurowanego klucza.

⚠️ Usunięcie PWA z ekranu głównego na iOS kasuje wszystkie dane. Regularnie rób eksport!

Wymagają zalogowania. Klucz VAPID jest pobierany automatycznie z serwera. Na iOS najpierw zainstaluj aplikację na ekranie głównym.
